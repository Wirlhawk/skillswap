# 🌟 SkillSwap

> Platform freelance khusus untuk siswa SMA/SMK yang ingin mengembangkan keterampilan mereka dengan mengambil pekerjaan (freelance project)

[![Live Demo](https://img.shields.io/badge/🌐_Demo_Website-skillswap--chi--two.vercel.app-blue?style=for-the-badge)](https://skillswap-chi-two.vercel.app)
[![Documentation](https://img.shields.io/badge/📑_Dokumentasi-Google_Docs-green?style=for-the-badge)](https://docs.google.com/document/d/1IwHq8k1y4Dd2i-x6_jsJ_rEgnD5It70jI39pivNPrKI/edit?tab=t.0)

**SkillSwap** adalah platform freelance khusus untuk siswa **SMA/SMK** yang ingin mengembangkan keterampilan mereka dengan mengambil pekerjaan (freelance project). Website ini membantu pelajar untuk belajar, bekerja, dan mengasah kemampuan mereka di bidang nyata.

---

## 🎯 Tujuan Proyek

1. **🎓 Meningkatkan pengalaman kerja pelajar**  
   Memberikan kesempatan siswa SMA/SMK untuk merasakan dunia kerja melalui freelance project.

2. **📝 Membangun portofolio sejak dini**  
   Membantu siswa mengumpulkan pengalaman dan karya nyata yang bisa digunakan untuk karir di masa depan.

3. **👨‍🏫 Menghubungkan siswa dengan klien**  
   siswa dapat mengambil pekerjaan yang sesuai dengan keterampilannya dan terhubung dengan klien.

## 🚀 Fitur Utama

### 🏠 **Beranda**

Halaman utama yang menampilkan layanan populer, testimoni pengguna, dan fitur-fitur unggulan platform.

### 🔍 **Pencarian & Penjelajahan Layanan**

Mencari dan menjelajahi berbagai layanan yang ditawarkan oleh pelajar dengan filter berdasarkan kategori, harga, dan rating.

### 💼 **Manajemen Layanan**

- **Pembuatan Layanan**: Pelajar dapat membuat dan menawarkan layanan mereka
- **Detail Layanan**: Menampilkan informasi lengkap tentang layanan termasuk deskripsi, harga, dan ulasan
- **Pengelolaan Layanan**: Pelajar dapat mengedit, menghapus, dan memperbarui layanan mereka

### 📋 **Portofolio**

- **Pembuatan Portofolio**: Pelajar dapat membuat portofolio untuk menampilkan karya-karya terbaik mereka
- **Showcase Portofolio**: Menampilkan portofolio pelajar kepada calon klien
- **Preview Portofolio**: Melihat pratinjau portofolio sebelum dipublikasikan

### 📊 **Dashboard Penjual (untuk pelajar)**

- **Ringkasan Kinerja**: Menampilkan statistik penting seperti pendapatan, jumlah pesanan, dan layanan populer
- **Grafik Pendapatan**: Visualisasi pendapatan berdasarkan periode waktu
- **Manajemen Pesanan**: Melihat dan mengelola semua pesanan yang masuk
- **Manajemen Layanan**: Mengelola semua layanan yang ditawarkan

### 🛒 **Manajemen Pesanan**

- **Pembuatan Pesanan**: Klien dapat memesan layanan dari pelajar
- **Detail Pesanan**: Melihat informasi lengkap tentang pesanan
- **Pengiriman Hasil**: Pelajar dapat mengirimkan hasil pekerjaan kepada klien
- **Milestone Pesanan**: Melacak kemajuan pesanan melalui milestone
- **Pesan & Lampiran**: Komunikasi antara klien dan pelajar dengan dukungan lampiran file
- **Ulasan Pesanan**: Klien dapat memberikan ulasan setelah pesanan selesai

### 👤 **Profil Pengguna**

- **Informasi Profil**: Menampilkan data pengguna dan statistik
- **Edit Profil**: Memperbarui informasi profil dan foto profil
- **Statistik Pengguna**: Menampilkan total pesanan, total pelanggan, ulasan, layanan, pendapatan, dan rata-rata nilai pesanan

### 💬 **Chat & Komunikasi**

- **Chat UI**: Antarmuka obrolan untuk komunikasi antara klien dan pelajar
- **Notifikasi**: Pemberitahuan untuk aktivitas penting dalam platform

### 🏆 **Leaderboard**

- **Rising Star**: Menampilkan pelajar berbakat yang sedang naik daun
- **Top 10**: Menampilkan 10 pelajar terbaik berdasarkan kinerja

### 🔐 **Autentikasi & Manajemen Pengguna**

- **Login & Registrasi**: Proses autentikasi pengguna
- **Manajemen Sesi**: Pengelolaan sesi pengguna yang aman

### 📱 **Halaman Pesanan Klien**

- **Daftar Pesanan**: Klien dapat melihat semua pesanan yang telah mereka buat
- **Filter Pesanan**: Memfilter pesanan berdasarkan status (Pending, Processing, Delivered, Cancelled)
- **Statistik Pesanan**: Melihat ringkasan status semua pesanan

### 🎓 **Dashboard Admin (untuk guru)**

- **Verifikasi Pelajar**: Guru dapat memverifikasi akun pelajar
- **Pemantauan Aktivitas**: Memantau aktivitas pelajar dan pesanan yang berjalan
- **Statistik Platform**: Melihat statistik keseluruhan platform

## 📦 Instalasi & Menjalankan Project

### 1. Clone repository

```bash
git clone https://github.com/username/SkillSwap.git
```

### 2. Masuk ke folder project

```bash
cd SkillSwap
```

### 3. Install dependencies

```bash
npm install
```

### 4. Setup database

```bash
# Import kategori dan jurusan ke database
psql -U username -d database_name -f categories-seed.sql
psql -U username -d database_name -f majors-seed.sql
```

### 5. Jalankan program

```bash
npm run dev
```

### 6. Buka di localhost

```
http://localhost:3000
```

## 📖 Cara Penggunaan

### 👨‍💼 Sebagai Klien (Buyer)

1. **Mendaftar/Login**
    - Gunakan email dan password untuk login atau daftar akun baru
    - Atau gunakan akun demo: `client@gmail.com` / `skillswap123`

2. **Mencari Layanan**
    - Jelajahi layanan di halaman beranda
    - Gunakan filter untuk menemukan layanan berdasarkan kategori atau harga
    - Klik pada layanan untuk melihat detail lengkap

3. **Memesan Layanan**
    - Pilih layanan yang diinginkan
    - Isi form pemesanan dengan detail kebutuhan Anda
    - Konfirmasi pesanan

4. **Mengelola Pesanan**
    - Pantau status pesanan di halaman "My Orders"
    - Komunikasikan dengan pelajar melalui fitur pesan
    - Terima hasil pekerjaan dan berikan ulasan setelah selesai

### 🎓 Sebagai Pelajar (Seller)

1. **Mendaftar/Login**
    - Gunakan email dan password untuk login atau daftar akun baru
    - Atau gunakan akun demo: `seller@gmail.com` / `skillswap123`

2. **Membuat Layanan**
    - Buka Seller Dashboard
    - Klik "Tambah Layanan Baru"
    - Isi detail layanan (judul, deskripsi, harga, dll)
    - Unggah gambar untuk layanan

3. **Mengelola Pesanan**
    - Terima atau tolak pesanan masuk
    - Komunikasikan dengan klien melalui fitur pesan
    - Kirim hasil pekerjaan melalui fitur pengiriman
    - Atur milestone untuk proyek yang lebih besar

4. **Mengembangkan Portofolio**
    - Tambahkan karya terbaik ke portofolio
    - Tampilkan hasil pekerjaan sebelumnya untuk menarik klien baru

### 👨‍🏫 Sebagai Guru (Admin)

1. **Mendaftar/Login**
    - Gunakan akun demo: `admin@gmail.com` / `skillswap123`

2. **Memantau Aktivitas**
    - Lihat statistik platform di Admin Dashboard
    - Pantau aktivitas pelajar dan pesanan yang berjalan

3. **Verifikasi Pelajar**
    - Verifikasi akun pelajar untuk memastikan kualitas
    - Tinjau portofolio dan layanan yang ditawarkan

## 🔐 Sistem Verifikasi Pelajar

SkillSwap memiliki sistem verifikasi untuk memastikan bahwa pelajar yang menawarkan layanan adalah siswa SMA/SMK yang sah:

1. **Proses Verifikasi**
    - Pelajar mengunggah foto kartu pelajar
    - Memilih jurusan dari daftar yang tersedia
    - Memasukkan nama sekolah

2. **Peran Pengguna**
    - **BUYER** (default): Pengguna baru yang dapat membeli layanan
    - **STUDENT**: Pengguna yang telah mengajukan verifikasi sebagai penjual
    - **TEACHER**: Penjual terverifikasi yang dapat menawarkan layanan dan meninjau permintaan verifikasi
